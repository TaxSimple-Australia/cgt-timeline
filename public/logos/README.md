# Logo Files Directory

## Instructions

Place your logo JPG files here with the following naming convention:

- `logo-1.jpg` - First logo option
- `logo-2.jpg` - Second logo option
- `logo-3.jpg` - Third logo option
- ... and so on up to `logo-10.jpg`

## Recommended Specifications

- **Format**: JPG (or PNG for transparency)
- **Dimensions**: 200-400px wide recommended
- **Aspect Ratio**: Maintain consistent proportions across all logos
- **Background**: White background for light mode logos, dark for dark mode

## Dark Mode Variants (Optional)

If you want different logos for dark mode, you can add:
- `logo-1-dark.jpg`
- `logo-2-dark.jpg`
- etc.

The system will automatically use the `-dark` variant when in dark mode if it exists.

## Updating Logo Metadata

Edit `/public/logos/logos-config.json` to update:
- Logo names
- Descriptions
- File paths (if using different naming)
